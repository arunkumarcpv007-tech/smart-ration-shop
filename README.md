## ğŸ“– Project Overview
**SmartPDS** is a single-page futuristic web application designed to digitize the ration distribution system. It eliminates queues, prevents fraud, and ensures transparency between the Government (Admin), Distribution Staff, and Citizens.

This project uses a **Serverless Architecture** (simulated via JavaScript State Management) to demonstrate the complete workflow in a browser environment.
